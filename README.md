
# Assai Dashboard

Dashboard de análise de visitantes para lojas Assai.

## Estrutura do Projeto

- `/server` - Backend API (Node.js + Express + PostgreSQL)
- `/src` - Frontend React (Vite + TypeScript + shadcn/ui)
- `/public` - Arquivos estáticos

## Scripts Disponíveis

### Desenvolvimento
```bash
# Frontend apenas
npm run dev

# Backend apenas
npm run backend:dev

# Ambos simultaneamente (recomendado)
npm run dev:full

# Backend em produção
npm run backend:start
